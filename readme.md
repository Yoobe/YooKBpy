# YooKBpy

Handwired keyboard code with :
- KMK firmware : (http://kmkfw.io/)
- RP2040-Zero board : (https://www.waveshare.com/wiki/RP2040-Zero)
- 0.95" RBG Oled : (https://www.waveshare.com/wiki/0.95inch_RGB_OLED_(B))
- 5 1.8mm Blue Leds
- One-handed custom layout
